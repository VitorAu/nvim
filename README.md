# Neovim Config

## Plugins
- telescope.nvim
- gruvbox.nvim
- conform.nvim
- nvim-treesitter
- toggleterm.nvim
- neo-tree.nvim
- lazygit integration
- lazy.nvim (plugin manager)

## External dependencies
Make sure these are installed on your system:

- Neovim >= 0.8
- Git
- ripgrep (`rg`)
- fd
- Node.js (for Treesitter)
- Lazygit

## Installation
Clone this config to `~/.config/nvim` and run Neovim.  
Lazy.nvim will automatically install all plugins on first launch.
